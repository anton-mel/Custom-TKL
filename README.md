An 84-key hotswap wireless mechanical TKL keyboard PCB, built around the nice!nano v2 (nRF52840, BLE 5.0 + USB-C), with MX-compatible hotswap sockets, diode-per-switch matrix, JST-PH LiPo battery input with SPDT power switch, 6x M2 mounting holes, and designed for ZMK. Ready to upload directly to JLCPCB, PCBWay, etc.
